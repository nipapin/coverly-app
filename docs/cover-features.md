# Coverly: что пользователь может делать с обложками

## Коротко

`Coverly` позволяет собирать обложки из готовых шаблонов: загружать свои картинки, менять текст, настраивать элементы, генерировать новые варианты через AI и экспортировать финальный результат.

---

## Типовой путь пользователя

1. Открыть проект и выбрать шаблон обложки.
2. Заполнить шаблон своими изображениями.
3. При необходимости сгенерировать новые варианты изображения.
4. Настроить текст и дополнительные элементы.
5. Сохранить результат и при необходимости вернуться к нему позже.
6. Экспортировать готовую обложку.

---

## Функции, доступные пользователю

### Выбор шаблонов
- Просмотр доступных проектов и шаблонов.
- Запуск редактирования из выбранного шаблона.
- Открытие ранее сохраненных работ из истории.

### Работа с изображениями
- Загрузка изображений перетаскиванием в зону.
- Загрузка изображений кликом по зоне.
- Выбор активного варианта из миниатюр.
- Удаление ненужных вариантов.
- Быстрая смена мест изображений между слотами (`Swap Images`).

### AI-генерация вариантов
- Генерация новых вариантов по текстовому описанию (`Generate`).
- Улучшение текста запроса кнопкой `AutoFix`.
- Автоматическая подгонка результата под область изображения.

### Позиционирование изображения
- Выравнивание по центру по горизонтали.
- Выравнивание по центру по вертикали.
- Подгонка по высоте.
- Подгонка по ширине.
- Ручное перемещение и масштабирование на холсте.

### Работа с текстом
- Редактирование текстовых блоков шаблона.
- Выбор шрифта.
- Изменение размера шрифта.
- Смещение текста вверх/вниз.

### Работа с декоративными элементами (assets)
- Переключение варианта элемента (например, стрелка/иконка).
- Изменение позиции элемента по слайдеру.
- Отражение по горизонтали и вертикали (в шаблонах, где это доступно).

### Управление редактором
- Масштаб холста (`Ctrl + колесо мыши`).
- Перемещение холста (`средняя кнопка мыши` или `Ctrl + ЛКМ`).
- Снятие/установка выделения элементов.

### Сохранение и история
- Сохранение работы кнопкой `Save`.
- Быстрое сохранение через `Ctrl+S`.
- Отмена и повтор действий (`Ctrl+Z`, `Ctrl+Y`, `Ctrl+Shift+Z`).
- Просмотр списка недавних работ и возврат к ним.

### Экспорт
- Экспорт готовой обложки в файл.

---

## Что важно учитывать

- Набор вкладок и инструментов зависит от конкретного шаблона.
- Некоторые шаблоны содержат только изображения, другие - изображения + текст + декоративные элементы.
- Инструменты перевода в интерфейсе присутствуют, но как основной рабочий сценарий для обложек сейчас используются не во всех шаблонах.
